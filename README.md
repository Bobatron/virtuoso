# Virtuoso

Virtuoso is a desktop tool for testing and debugging XMPP stanza messages, inspired by Postman but focused on XMPP workflows. It allows users to connect to XMPP servers as multiple accounts, send custom XML stanzas, and view responses in real time.

## Features
- Multi-account support
- Paste/send XML stanzas
- Real-time message display
- Account management

## Technology
- Electron, Node.js, React, @xmpp/client, TypeScript

# Virtuoso Notes

The right side will have 2 tabs
Composer / Conductor

Composer will be able to record a composition
There will be a list of existing compositions
The composer can delete them
the composer can view them
the composer can click on stanzasf from a composition to load them into the UI
the composer can remove actions, stanzas from a composition
the composer can create new compositions
the composer can record onto existing compositions
the composer can shuffle the order in existing compositions

the conductor
Can export compositions as performances
Can play compositions as performance
Can view performance results
